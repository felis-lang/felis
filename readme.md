# Felis

个人学习 Rust 时用于练手实现的编程语言。非常简单，适合想要自己设计语言的人参考。

## 语法特性

### 已实现特性

- arithmetic expression
- first-class function
- variable binding
- type binding (暂时不能用于子类型检查，等待 recursive types 实现)
- if expression
- block expression

### 计划实现

- recursive types
- mutable variable (?)
- while expression
- return statement
- enum / match
- universal type / inference
- HKT
- module
- trait / operator overloading

### 可能不会实现

- record / interface
- dependent type
